# 2021 Recap

A slideshow summarizing key metrics and milestones from 2021, covering:

- **Google Analytics** — weekly user, web dashboard, and mobile dashboard trends (2019 vs 2020 vs 2021)
- **App Store** — Android and iOS ratings (2019 vs 2020 vs 2021)
- **Search Console** — weekly impressions, click rate, and clicks (2019 vs 2020 vs 2021)
- **Web Servers** — weekly request volume and traffic
- **Status** — outage duration and uptime (2019 vs 2020 vs 2021)
- **Database** — shift, assignment, absence, and mail counts (2018–2021)
- **System** — total users and shift hours
