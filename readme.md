# Lets Talk about ... Whats new (and upcoming) in JS (2019 Edition)

[View Slides](https://new-js-features--2019.lets-talk-about.tamino.dev)

## Summary

A walkthrough of JavaScript features across the TC39 proposal stages (0–4), covering nested imports, Math extensions (clamp, scale, seeded PRNG), iterator helpers, Observables, Promise additions (try, any, allSettled), decorators, Temporal, Set/Map extensions, throw expressions, Realms, optional chaining, nullish coalescing, numeric separators, class field enhancements (static, private), optional catch binding, Intl formatting APIs, and Array/Object extensions (flat, flatMap, fromEntries).
