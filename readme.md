# Lets talk about - API Workshop #1

A workshop covering the fundamentals of building an API with Rails, from database design to documentation. The slides walk through four core layers:

- **Database** — Migrations, schema, seeds, views, functions, and triggers
- **Models** — Relationships, callbacks, scopes, validations, and helpers
- **Operations** — Extracting business logic out of models and controllers into standalone operation objects (initialize, call, helpers)
- **API** — Shared helpers, entry points, endpoint design (parameters, actions, resource workflow), and documentation with Swagger/OpenAPI
