- **Jira** — release count growth
- **Analytics** — weekly user, startpage, and mobile dashboard trends (Jul–Dec)
- **Search Console** — daily search volume for "shyftplan" (2018 vs 2019)
- **Cloudfront** — weekly request volume, error rates, resilience during outages, and uptime
- **System** — staff shift, absence, and user counts (year-over-year and totals across 131 countries)
