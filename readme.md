# Lets Talk about ... Whats new (and upcoming) in JS (2020 Edition)

[View Slides](https://new-js-features--2020.lets-talk-about.tamino.dev)

## Summary

A slideshow covering new and upcoming JavaScript features as of 2020, organized by TC39 proposal stage:

- **Stage 0/1 (Strawperson/Proposal):** Dedent template literals, `randomUUID()`, async constructors, `Number.range` / `BigInt.range`, Array extensions (`equals`, `uniqueBy`, `item`), and `await.ops` shorthand
- **Stage 2/3 (Draft/Candidate):** Private class fields (`#`), Record & Tuple immutable primitives, import assertions, optional chaining (`?.`), nullish coalescing (`??`), numeric separators (`1_000`), and top-level `await`
