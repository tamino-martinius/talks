# Lets Talk about ... Whats new (and upcoming) in JS (2018 Edition)

[View Slides](https://new-js-features--2018.lets-talk-about.tamino.dev)

## Summary

A slide deck covering modern JavaScript features introduced around ES6 and beyond, including arrow functions, promises, async/await, rest parameters, template literals, iterators, generators, and async generators. Each topic is presented with side-by-side ES5 vs ES6+ comparisons and common pitfalls to watch out for.
