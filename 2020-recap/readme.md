# 2020 Recap

A slideshow summarizing key metrics and milestones from 2020, covering:

- **Google Analytics** — weekly user, web dashboard, and mobile dashboard trends (2019 vs 2020)
- **App Store** — crash counts and ratings for Android and iOS
- **Search Console** — weekly impressions, click rate, and clicks (2019 vs 2020)
- **Web Servers** — weekly request volume, traffic, and peak load handling
- **Status** — outage duration and uptime
- **Database** — shift, assignment, and absence counts (2018 vs 2019 vs 2020)
- **System** — total shift hours and mails sent
